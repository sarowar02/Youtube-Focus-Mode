# Youtube-Focus-Mode
A chromium based browser extension to block youtube home feed and side bar 

## SETUP
1.Download the project folder or git clone to your machine <br>
2.Open Your Browser <br>
3.Go to Extension Manager  <br>
4.Enable Developer Mode <br>
![developer_mode](developer_mode.png) <br>
5.Click on Load Unpacked <br>
6.Give the Location of the folder <br>
7.Now go to Extensions and Open Youtube Focus Mode <br>
8. Click on turn on thats it <br>
![youtube](youtube.png) <br>
